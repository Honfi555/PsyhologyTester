# PsyhologyTester
 
Файлы с анкетами и отчётами расположени по пути C:\Tests_Psychology

Инструкция:
1. Заполнение файлов с тестами
	1 строка - Название теста
	2 строка - Тип теста(Опросник/Закончи предложение)
	3 строка - Вопрос
	4 строка - Варианты ответов(Между каждым выриантом ответов должно находиться ключевое слово next)
	Остальные строки пишутся по аналогии с 3 и 4 строкой.
2. Для выбора теста из списка внутри программы нажнимете на кнопку "Выбрать", если тест не выьран и кнопка нажата откроется 1 тест
3. Для добавления теста в список необходимо нажать на кнопку "Добавить" и выбрать файл с тестом, если после добавления теста, файл с ним переместить тест не будет находиться
